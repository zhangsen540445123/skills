# 《只管去做》核心方法论

> 如何找到人生方向，制定并实现年度目标

## 核心理念

**30岁左右的焦虑，往往是因为没有按照自己想要的方式活着！**

## 四种人生状态

根据**掌控**和**视角**两个维度，人生分为四种状态：

### 1. 空想家（高视角 + 低掌控）
- 总是有很多想法，但真正落实去做的很少
- 不停抱怨现状，但从不改变
- 典型表现：每次聚会都说要创业，但一直在原公司

### 2. 回应者（低视角 + 低掌控）
- 对未来没有任何想法，对现状也没有掌控力
- 混吃等死的状态，每一天都过得差不多
- 被动地回应外界刺激

### 3. 执行者（低视角 + 高掌控）
- 能力很强，交代的事情都能搞定
- 但整天忙于琐事，没有时间考虑自己的未来
- 像戴着眼罩狂奔的马

### 4. 舰长和指挥官（高视角 + 高掌控）⭐
- 对未来有清晰规划，又有足够能力实现
- 最有幸福感和成就感
- **不管有钱没钱、当官不当官，都可以进入这种状态**

## 年度计划失败的四大原因

### 假 - 目标不是发自内心的
- 一时激情，而非真正想要
- 看到别人做什么就跟风
- 没有让自己怦然心动的愿景

### 大 - 目标超出能力范围
- "每天走一万步"不难，难的是"每天"
- 要求自己每一天都做同一件事，绝对超出能力范围

### 空 - 写下目标但没有落实到计划
- 年度目标没有拆解到每个月、每一天
- 到年底才想起来还没完成

### 全 - 贪多求全
- 同时想学很多东西
- **如果你有很多目标，说明你还不知道自己究竟想要什么**

## 核心方法：飞机落地法

### 第一步：写给五年后自己的一封信

**去脑存心法**：
1. 不要用大脑评判，而要用心感受
2. 调动五感：视觉、听觉、嗅觉、味觉、触觉
3. 写出让自己怦然心动的画面

**关键问题**：
- 五年后的你在哪里？
- 在做什么？
- 和谁在一起？
- 周围的环境是什么样的？
- 你的感受是什么？

### 第二步：愿景孵化成目标

**愿景 vs 目标**：
- **愿景**：激励人心的画面，刺激右脑，让人兴奋，但无法直接行动
- **目标**：明确具体的结果，刺激左脑，可以分析和计划

**GPS方法**（三步孵化）：
1. **G - Goal（定义它）**：怎么算达成？可验证的成果画面
2. **P - Person（找关键人）**：谁决定你是否达成？
3. **S - Standard（定标准）**：具体的衡量标准是什么？

**案例**：
- 愿景："项目审查水平提高"
- 目标："通过率从60%提高到80%"
- 关键人：评审专家
- 标准：连续3个月通过率≥80%

### 第三步：检查目标是否合理

**时间资源盘点**：
- 想做的事就像湖水，能搞定多少由舀水的杯子决定
- 杯子大小 = 时间资源多少
- 先盘点时间资源，再设定目标

**合理的计划**：
- 合乎道理（规律、环境、他人）
- 合乎情理（能力、精力、可控时间）
- 不是考虑"完美的自己"，而是"真实的自己"

### 第四步：甘特图（抬头看路）

**作用**：
- 让你看到全局
- 明确里程碑
- 避免"低头走路"忘记方向

**关键**：
- 不要太细，只标注关键节点
- 定期回看，调整方向

### 第五步：倒推分解法（低头走路）

**从年度目标到今日行动**：
1. 年度目标 → 季度里程碑
2. 季度里程碑 → 月度计划
3. 月度计划 → 周计划
4. 周计划 → 今日第一步

**关键原则**：
- 每层都要有"完成定义"
- 最后只保留"今天能开始的第一步"

## 习惯养成方法

### 三分钟热度的本质

不是意志力不够，而是：
1. **没有燃料**：缺少内在动力
2. **计划不坚韧**：追求完美，一次失败就放弃

### 习惯养成的燃料

**三种燃料**：
1. **成就感**：看到进步
2. **社交支持**：有人陪伴
3. **意义感**：知道为什么做

### 坚韧的计划 > "一定要"的计划

**三条命机制**：
- 一个月允许3次失败
- 用完3次才算真正失败
- 类似游戏，更符合人性

**习惯卡片**：
- 记录每天完成情况
- 可视化进度
- 及时反馈

## 每日4D工作法

### 不是没时间，是三个问题

1. **不是没时间，是没精力**
   - 早晨精力旺盛，下班后像行尸走肉
   - 要在精力最好的时候做最重要的事

2. **不是没时间，是没大块时间**
   - 时间被切割得支离破碎
   - 圆球放不进碎片里

3. **不是没时间，是没有立刻去做**
   - 想到了就立刻去做
   - 避免被干扰打断

### 4D工作法

**左栏（时间轴）**：
- 标注固定时间的事情
- 看清真正可用的时间资源

**右栏（自由区）**：
- 真正的待办事项
- 标注重要事项（星号）
- **必须包含年度计划的事情**

**关键原则**：
- 忙完工作再去实现年度计划就没戏了
- 年度计划的事情要写进每日清单
- 在精力最好的时候做最重要的事

## 适用场景

- ✅ 不知道自己想要什么
- ✅ 年度计划总是实现不了
- ✅ 有很多想法但不知道从哪开始
- ✅ 习惯总是坚持不下来
- ✅ 每天很忙但没有为自己而活
