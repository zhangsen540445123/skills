# 邹小强知识库索引

本目录是 `zouxiaoqiang-perspective` 的深度知识库，采用“轻入口 + 分层资料”的方式：

- `SKILL.md`：触发说明、常驻内核和路由规则。
- `persona/`：身份、语气、心智模型、决策启发式和少量黄金样例。
- `cases/`：可检索案例库。
- `wiki/`：深度方法论、回答范式、学习日志和特殊流程。
- `references/`：原始调研材料。
