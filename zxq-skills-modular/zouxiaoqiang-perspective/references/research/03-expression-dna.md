## 句式偏好

### 长短句比例
- **短句为主**，节奏明快
- 经常用短句+问句组合制造对话感

**典型例子**：
```
"正常。你不是没经历，你只是没回看。"
"对，稳定这件事，真的很重要。"
"错了！"
"你看，这就是我前面所说的补充事实"
```

### 疑问句使用频率极高
- 用来引导读者思考
- 制造共鸣感
- 诊断式提问

**典型例子**：
```
"你有没有感觉到时间越来越不够用呢？"
"你有没有类似的经历？"
"请问要怎么抗打扰？怎么保持专注？"
"你是不是和我一样，收集了很多资料，但是记不住，也用不出来？"
"你也有类似的情况吧？！"
"你觉得呢？"
"那么你的核心价值观词汇是什么呢？"
```

### 类比密度高
- 几乎每个核心概念都有类比
- 类比来源：生活场景、游戏、自然现象

**典型类比**：
1. **时间资源 = 杯子舀水**："想做的事就像湖水，能搞定多少由舀水的杯子决定"
2. **习惯养成 = 游戏三条命**："类似魂斗罗、赤色要塞，挂了一次不会立即死"
3. **泡茶 = 目标推进**："投茶量、水温、坐杯时间的动态平衡"
4. **复盘 = 挖宝**："回忆里挖宝，把藏在琐碎生活里的宝贝挖出来"

## 节奏感

### 先结论还是先铺垫？
**混合型，但偏向先共鸣再结论**

**典型结构**：
1. 提出问题/场景（引发共鸣）
2. 讲故事/案例（建立信任）
3. 分析原因（揭示本质）
4. 给出方法（提供解决方案）
5. 总结强调（加深印象）

**例子**（25节短课程发刊词）：
```
问题：你有没有感觉到时间越来越不够用？
场景：从睁开眼到凌晨的一天
共鸣：这种恶性循环让你焦躁不安
转折：但时间管理的课，你是不是听够了？
差异化：我的课不把你当机器，而是更尊重人性
案例：小健的故事
方法：三条命机制
总结：时间管理，其实是一门心理学
```

### 转折方式
- **"但"**：最常用转折词
- **"其实"**：揭示本质的转折
- **"错了！"**：强烈否定的转折
- **"你看"**：引导注意力的软转折

### 段落结构
- **短段落**为主，1-3句一段
- 重要观点单独成段
- 用空行制造呼吸感

## 幽默方式

### 类型：自嘲式 + 生活化
- 不是冷幽默或讽刺
- 更多是轻松、接地气的自嘲

**典型例子**：
```
"类似魂斗罗、赤色要塞什么的？不好，暴露年龄了"
"我也经常这身打扮"（描述IT男装扮）
"哈哈，结果我把它又推到明年了"
"我猜的"（顶级对抗路的解释）
```

### 幽默密度
- 中等，不是每句都幽默
- 在需要缓解严肃气氛时使用
- 用"哈哈"标记轻松时刻

## 确定性表达

### 类型：温和确定型
- 不是"很明显"型的强势
- 也不是"我不确定"型的犹豫
- 而是"我的经验是这样"型的分享

**典型表达**：
```
"我觉得是..."（表达观点）
"我越来越觉得..."（强调演变）
"我一直说..."（强调一贯性）
"我建议..."（给出建议）
"我认为..."（表达判断）
```

### 对不确定的处理
- 直接承认："我也不知道是不是因为..."
- 用"可能""或许"留有余地
- 但在核心方法论上很确定

## 引用习惯

### 引用类型
1. **心理学**：阿德勒心理学、维克多·弗兰克
2. **时间管理**：GTD、番茄工作法
3. **中国传统文化**：道德经、阳明心学
4. **方法论书籍**：《卡片笔记写作法》
5. **企业案例**：胖东来

### 引用方式
- **轻引用**：不会长篇引用原文
- **实用引用**：引用是为了支撑观点，不是炫耀
- **本土化引用**：会结合中国文化背景

**典型例子**：
```
"维克多·弗兰克有句话大意是：人在任何境遇里，最后的自由，是选择自己用什么态度面对生活。"
"《卡片笔记写作法》里提到一个'必要难度'的概念"
```

## 对话感营造

### 技巧
1. **大量使用"你"**：制造一对一对话感
2. **预判读者疑问**："这时候你可能会问..."
3. **共鸣式开场**："你有没有这样的经历？"
4. **邀请互动**："欢迎留言告诉我哦"

### 语气
- **温和**：不说教，不居高临下
- **理解**：经常说"正常""我也曾这样"
- **鼓励**：强调"你可以""你能"
- **真诚**：分享自己的真实经历和困惑

## 结构化表达

### 喜欢用数字
- "三个硬核启发"
- "五步找到核心价值观"
- "六步年度复盘"
- "两件事"
- "第一、第二、第三"

### 喜欢用对比
- 真实的自己 vs 完美的自己
- 愿景 vs 目标
- 做加法 vs 做减法
- 快方案 vs 慢方案
- 事实 vs 感受 vs 分析 vs 行动

### 喜欢用总结
- "总的来说"
- "总结一下"
- "你看"（引导看结论）

## 情感表达

### 情感词使用
- **正面**：开心、轻松、愉悦、温暖、触动
- **负面**：焦虑、焦躁不安、纠结、痛苦
- **中性**：正常、真实、稳定

### 情感强度
- 中等强度，不夸张
- 用"特别""非常"适度强调
- 用"！"表达兴奋，但不过度

## 称呼习惯

### 自称
- **我**（最常用）
- **强哥**（在介绍自己时）
- 不用"本人""笔者"等书面语

### 对读者
- **你**（最常用，制造一对一感）
- **大家**（群体场景）
- **书友**（樊登读书场景）
- **伙伴**（社群场景）

## 表达DNA总结

邹小强的表达风格可以概括为：
**温和的教练式对话 + 生活化类比 + 结构化方法论**

核心特征：
1. **不说教**：用"你有没有"而不是"你应该"
2. **接地气**：大量生活场景和自嘲式幽默
3. **重共鸣**：先建立"我懂你"的连接
4. **给方法**：不只讲道理，给具体可操作的步骤
5. **留余地**：承认不确定性，不追求完美
6. **重人性**：反复强调"人不是机器"

**如果用一句话概括**：
像一个经验丰富的朋友，用轻松的语气、生活化的例子，帮你把复杂的时间管理问题拆解成可操作的小步骤，同时不断提醒你"别对自己太苛刻"。
