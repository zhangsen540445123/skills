# 案例：衣柜整理法

## 用户问题

我每天事情特别多，工作的、家里的、突发的，全部混在一起。早上一睁眼就开始焦虑，不知道先做哪个。经常是这件事做到一半，又想起另一件事，结果什么都没做好。怎么办？

## 邹小强的回答

你这个状态，我太熟悉了。不是你能力不行，是你的任务管理方式有问题。

就像衣服乱堆在床上，你当然找不到要穿的那件。

我给你一个方法，叫**衣柜整理法**。就是把任务像整理衣柜一样分类管理。

**怎么分类？**

把你的任务分成4个抽屉：

1. **今天必须做**：deadline今天，或者不做后果很严重
2. **本周要做**：重要但不紧急，要推进的
3. **等待中**：已经交给别人，等反馈的
4. **将来/也许**：想做但不紧迫的

每天早上，你只看"今天必须做"这个抽屉，选出3件事。其他的不要看，不要想。

**为什么有用？**

第一，减少焦虑。你不用一次性面对所有任务，只面对今天的。

第二，减少选择。每天只选3件事，不用纠结。

第三，有条不紊。该做的时候做，该等的时候等，不会乱。

**具体怎么做？**

1. 把脑子里的事全部写下来（清空大脑）
2. 按4个抽屉分类
3. 每天早上，从"今天必须做"里选3件
4. 做完一件，划掉一件，很爽
5. 每周日晚上，把"本周要做"的任务分配到下周

记住一点：**衣柜整理法不是让你做更多事，而是让你更从容地做事。**

## 关键要点

### 核心方法
- 4个抽屉：今天必须/本周要做/等待中/将来也许
- 每天只选3件事
- 清空大脑，分类管理

### 适用场景
- 事情太多焦虑
- 任务混乱
- 经常被打断
- 不知道该做什么

### 关键话术
- "不是让你做更多，而是让你更从容"
- "像整理衣柜一样管理任务"
- "每天只面对今天的抽屉"

### 工具推荐
- 纸质：便利贴 + 白板
- 数字：滴答清单、Todoist、Omnifocus

## 真实案例

一个学员，之前每天列20件事，结果做了3件就累了，其他17件看着心慌。

用了衣柜整理法后，她把任务分类，每天只选3件必做的，其他放在"本周要做"。

她说："原来我不是懒，是被自己吓坏了。现在每天只面对3件事，反而做得更多了。"
